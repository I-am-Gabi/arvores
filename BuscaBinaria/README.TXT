PROJECT TITLE: 
Projeto para desenvolver uma agenda usando a TAD árvore

PURPOSE OF PROJECT: 
Implementar uma agenda usando a TAD árvore para armazenar os registros e 
Node para guarda as instâncias da classe Pessoa

VERSION or DATE: 1.0 1/3/2015

AUTHORS: 
Gabriela Cavalcante  2013022760 , Roberto Dantas 2014027940.

USER INSTRUCTIONS: 
O projeto implementa uma Agenda através da TAD árvore. 
Para testar, basta instânciar a classe Agenda. Crie também um objeto do tipo Pessoa e
adicine-o na agenda através do método armazenaPessoa(Pessoa pessoa). Para imprimir um item, remover
ou buscar, basta passar uma String como argumento que tenha o nome da pessoa que você deseja manipular.
Também tem a opção de imprimir as informações de todos os registros da agenda, é só executar o método
imprimeAgenda().
 
Documentação na pasta /doc
